Zipper

Zipper is a full-featured asset packaging plugin for Maven that provides JavaScript lint checking, Closure Compiler and YUI Compressor support, custom CSS and JavaScript concatenation, and up-front gzipping.

Getting Started

  1. Add the following plugin definition to the build section of your POM file: 
    <plugin>
  <groupId>com.joestelmach<groupId>
  <artifactId>zipper<artifactId>
  <version>0.1<version>
  <configuration>
  ...
  <configuration>
<plugin>
  2. Populate the configuration section as needed.
  3. Execute the pack goal directly with 
    mvn com.joestelmach:zipper:pack
    or bind the goal to the process-resources phase by adding the following execution to the above plugin section: 
    
<executions>
  <execution>
    <goals>
      <phase>process-resources<phase>
      <goal>pack<goal>
    <goals>
  <execution>
<executions>
  4. Enjoy faster page loads
